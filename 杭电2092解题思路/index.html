把x表示为两个完全平方数的和，即x=a^2+b^2，其中a,b为整数。
是完全平方数的和，当且仅当x为偶数，且x的平方根为整数。
先判断是否为完全平方数，再判断是否为两个完全平方数的和。

后发现时间超出限制，改变思路

可使用韦达定理，即(a+b)^2=a^2+2ab+b^2。
因此，x=a^2+b^2，其中a,b为整数，当且仅当x为偶数，且x的平方根为整数。
根据韦达定理，x的平方根为整数，当且仅当x为偶数。
因此，x为偶数，且x的平方根为整数，当且仅当x为完全平方数的和。

根据通用求根公式求根需要满足条件如下：
1.Δ = n^2 - 4m ≥ 0

2.sqrt(Δ) 是整数，即 Δ 是完全平方数

3.(n + sqrt(Δ)) 是偶数（等价于 n 和 sqrt(Δ) 同奇偶）

推导 s 和 d 同奇偶性:

由Δ = s² - 4p = d²，得 s² - d² = 4p，即 (s - d)(s + d) = 4p。

因为两数之和 2s 为偶数，故 s - d 与 s + d 同奇偶

又因为它们的乘积是偶数（4 的倍数），所以 s - d 和 s + d 均为偶数，

从而 s 和 d 同奇偶。

因此只需要确保n^2 - 4m ≥ 0，且sqrt(n^2 - 4m)为整数即可。
